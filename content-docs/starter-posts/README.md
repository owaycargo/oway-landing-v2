# Стартовые посты для блога OWAY CARGO

5 готовых статей-шаблонов для публикации в Strapi. Написаны с соблюдением
`content-guide.md` — tone of voice, структура, SEO.

## Как использовать

1. Зайдите в Strapi админку (адрес вашей CMS)
2. Содержание → News Item → Create new entry
3. Скопируйте содержимое .md файла в поля:
   - `title` — H1 заголовок
   - `title_seo` — SEO-заголовок (отдельный если отличается от H1)
   - `seo_description` — meta description
   - `card_description` — короткое описание для карточки
   - `description` — полный текст статьи (markdown)
   - `slug` — URL slug
4. Выберите категорию (news_types)
5. Publish
