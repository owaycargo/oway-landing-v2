# OWAY CARGO — настройка аффилиат-ссылок

Как подключить партнёрские программы магазинов для получения % с покупок клиентов.

## Зачем

Когда клиент OWAY идёт по ссылке на Amazon/eBay и покупает — магазин платит нам комиссию (1-10% стоимости покупки). Плюс клиент оставляет посылку у нас на доставку. Двойная монетизация одного трафика.

## Архитектура

Файл `lib/affiliate.ts` автоматически оборачивает исходящие ссылки. Если env-переменная не задана — возвращает исходный URL. Не требует изменений в коде при подключении/отключении.

Интегрирован в:
- `/marketplace/[id]` — кнопка «Перейти на сайт магазина»

Планируется добавить:
- Ссылки в контенте блог-постов (когда появятся)
- eBay flagship guide
- Category landing pages

## 1. Amazon Associates

### Регистрация
1. https://affiliate-program.amazon.com/ → Join Now
2. Указать данные: имя, адрес, налоговый ID (для США — SSN/EIN; для других стран — W-8BEN форма)
3. Указать сайт: `owaycargo.com`
4. Получить **Associate Tag** (обычно формат `youraccountname-20`)

### Настройка
**Railway** (production):
```
NEXT_PUBLIC_AMAZON_ASSOCIATES_TAG=owaycargo-20
```

**Vercel** (preview):
```
NEXT_PUBLIC_AMAZON_ASSOCIATES_TAG=owaycargo-20
```

### Требования Amazon
- Должно быть минимум 3 продажи в первые 180 дней — иначе аккаунт закроют
- Раскрытие «As an Amazon Associate I earn from qualifying purchases» — добавить на сайт (рекомендуется в подвал или политику)
- Нельзя использовать для email-ссылок на Amazon
- Один account tag per website

### Комиссии
- Fashion (одежда, обувь) — 4%
- Home & Garden — 3%
- Electronics — 2-3%
- Amazon Private Labels — 10%
- Amazon Prime Video — 5%

Ориентир: средний чек $100, клиент покупает одежду → $4 комиссия за каждую покупку.

## 2. eBay Partner Network (EPN)

### Регистрация
1. https://partnernetwork.ebay.com/ → Apply Now
2. Указать сайт, тип аудитории, стратегию
3. Получить **Campaign ID** (`campid`) — 10-цифровой номер

### Настройка
**Railway + Vercel**:
```
NEXT_PUBLIC_EBAY_CAMPAIGN_ID=5338123456
```

### Особенности EPN
- Платят за продажи (commission) и за новых клиентов (bonus)
- Комиссия 50-70% от платы eBay продавцу за первые 24 часа
- В среднем — $5-15 с покупки iPhone Б/У
- Есть API tools (продвинутый таргетинг) — для будущего использования

### Требования
- Сайт должен генерить качественный трафик (eBay периодически проверяет)
- Нельзя покупать траф на свой сайт
- Нужна страница с условиями (уже есть — /privacy, /terms)

## 3. Другие программы (будущее)

Когда появится трафик — можно подключать:
- **Target Partner** — 1-8% commission
- **Walmart Affiliate Program** — 1-4%
- **iHerb Rewards** — reward codes для клиентов
- **Skimlinks / Viglink** — универсальные агрегаторы (автоматически подключают десятки магазинов)

## Как проверить что работает

1. Открыть `/marketplace/amazon` на owaycargo.com
2. Нажать «Перейти на сайт магазина»
3. В URL-баре должно быть `?tag=owaycargo-20` (или ваш tag)
4. Сделать тестовую покупку (лучше дешёвой вещи)
5. Через 24-48 часов проверить в affiliate dashboard — должна появиться транзакция

## Учёт доходов

Amazon платит раз в ~60 дней, минимальная выплата $10 (Direct Deposit) или $100 (чеком).
eBay платит ежемесячно, минимум $10 PayPal.

## Важно: раскрытие

По правилам FTC (США) и аналогам в других странах — нужно раскрывать что получаете комиссию с покупок.

Рекомендуется:
1. В `/privacy` добавить абзац «Affiliate disclosures»
2. На странице `/marketplace/[id]` — мелкий текст под кнопкой «Партнёрская ссылка» (можно)
3. В футере — общая ссылка на disclosure

Пример текста:
> «Некоторые ссылки на сайте — партнёрские. Если вы перейдёте по ссылке и совершите покупку, OWAY CARGO может получить комиссию. Это не влияет на цену товара для вас.»

## Trackable events

Клики по affiliate-ссылкам уже трекаются в GA4:
- `shop_visit` с параметром `affiliate: yes/no`

В GA4 можно будет видеть какой % трафика уходит по affiliate-ссылкам → корреляция с доходом с dashboard Amazon/eBay.
